# **App Name**: Culinary Canvas

## Core Features:

- Recipe Creation & Upload: Allow users to create and upload recipes with ingredients, instructions, and photos.
- Community Feed: Display a feed of shared recipes from other users with options to sort and filter.
- Personal Cookbook: Enable users to save their favorite recipes to a personal cookbook for easy access.
- Recipe Interaction: Allow users to interact with recipes through comments, ratings, and reviews.

## Style Guidelines:

- Background color: Creamy white (#F8F5F0) for a warm and inviting feel.
- Primary color: Soft terracotta (#E07A5F) to evoke a sense of home cooking and comfort.
- Accent color: Sage green (#A7C4A3) for a fresh and natural touch, complementing the terracotta.
- Body font: 'Literata', a serif font, with a literary, vintage, slightly formal feel
- Headline font: 'Playfair', a modern sans-serif, similar to Didot, geometric, high contrast thin-thick lines, with an elegant, fashionable, high-end feel.
- Use clean, simple icons representing different food categories and actions like saving, sharing, and commenting.
- Design a user-friendly layout with clear sections for recipe details, user profiles, and community feed.
